# 中古車販売店LP（制作サンプル）

架空の中古車販売店「オートガレージ」の8ページ構成ランディングページ。ポートフォリオ用の制作サンプルです。

🔗 **公開URL:** https://sleepycat12341013.github.io/used-car-lp/

## 構成
- HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- 8ページ：トップ / 在庫一覧 / 車両詳細 / 選ばれる理由 / ご利用の流れ / 店舗紹介 / 来店予約 / 在庫問い合わせ
- レスポンシブ対応（スマホはハンバーガーメニュー）
- ヒーロー画像スライドショー、在庫のフィルター・ページャ・自動切替（デモ）

## 注意事項
- 架空の店舗・ダミーデータです。電話番号・住所・在庫はすべて架空。
- フォームは送信されません（デモ表示のみ）。
- 使用画像にはAI生成のものを含みます。
